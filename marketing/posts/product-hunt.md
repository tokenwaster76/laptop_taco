## Description

Laptop Taco is a single Bash script that keeps your Mac awake while a long-running command finishes. Built for the new normal of AI coding agents that take 10–30 minutes to refactor a repo.

It wraps your command with macOS `caffeinate`, prints PID and battery, warns you on low battery, and sends a native notification when the agent is done. Exits with the same code as the wrapped command.

No app. No account. No cloud. No dependencies. One file, MIT licensed, macOS-only.
