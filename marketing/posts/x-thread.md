**Post 2**

It's one Bash script. macOS-only. Zero deps. It wraps your long-running agent command with `caffeinate` and tells you when the agent is done cooking.

No app.
No account.
No cloud.
No subscription.
No newsletter.
Just a responsible taco.

---

**Post 3**

Usage is the obvious thing:

./taco claude
./taco codex
./taco "npm test"
./taco "rake spec"

It runs the command, keeps your Mac awake until the process exits, and then exits with the same code your command returned.
